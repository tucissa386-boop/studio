# **App Name**: MindType

## Core Features:

- Typing Sprint: Users type freely for 10 seconds, capturing raw thoughts.
- Real-time Analysis: Analyzes text input during the typing sprint to identify emerging patterns and sentiment.
- Thought Dissection: Breaks down the text into micro-thoughts to reveal hidden insecurities, hopes, thought habits, and negative self-talk.
- Pattern Highlighting: Highlights key psychological patterns within the text, making them easily identifiable. It also gives insight into potential impact on behavior and relationships
- Session History: Allows users to review past typing sprints and track changes in their thought patterns over time.
- AI-Powered Recommendations: Offers personalized recommendations based on the identified patterns. The recommendations might point to other exercises the user can engage with.
- Data Visualization: Offers custom charts that visualise data from thoughts.

## Style Guidelines:

- Primary color: Deep violet (#4B0082) to evoke introspection and calmness.
- Background color: Off-white (#F5F5F5) for a clean and non-distracting environment.
- Accent color: Soft lavender (#E6E6FA) to highlight key insights.
- Body and headline font: 'Literata', serif, providing a readable and classic feel for longer blocks of text.
- Use simple, minimalist icons to represent different psychological insights. Avoid overly complex designs.
- Ensure a clean and distraction-free layout, maximizing focus on the typing area and analysis results. Prioritize readability.
- Subtle animations to guide the user's attention without being intrusive. For example, a gentle pulse on the start button.
